# Hung Up On Hummus

## The Origin Story
Every so often (which is more often than I should admit) I catch myself dreaming of East Tea Can's Legendary Hummus + Chicken (A most apt name, if I do say so myself). 

> [!TIP]
> **If you've never tried it before:** Close this tab and [order some](https://order.toasttab.com/online/east-tea-can-new-3115-winston-churchill-blvd-unit-1/item-legendary-hummus-chicken_2c74bb69-ea49-4faf-99c9-e0fc70f86bcc) ASAP!


Last summer, I discovered the most amazing promotion known to mankind - 🌟THE LEGENDARY BOGO DEAL 🌟. So now, my morning ritual includes waking up and refreshing East Tea Can's online ordering site, daring to hope that the deal will be on today.
 <img width="261" height="250" alt="aq0afo" src="https://github.com/user-attachments/assets/b17a883a-3365-4ead-9c15-f54cff0f523a" />


Due to the unpredictable nature of the deal, I'm now delegating this critical task to a Python bot!

## Tech Stack 
The script is pretty simple. It uses:
- Playwright
- SMTPLIB (for email notifications)
- GitHub Actions (for continuously running the script on a schedule)
